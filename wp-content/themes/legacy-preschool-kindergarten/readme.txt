== Description ==

Legacy Preschool Kindergarten is a multipurpose theme designed for early education websites such as preschools, kindergartens, daycare centers, and child-focused learning programs. The theme features a clean and minimal layout with a responsive design that adapts to desktops, tablets, and mobile devices. It includes sections commonly used on educational websites, such as a banner area, testimonial section, staff/team showcase, and call-to-action buttons. Social media integration is available, and the theme is translation-ready for multilingual use.Developers and site owners can customize layouts, colors, and content using built-in options and shortcodes. The theme uses Bootstrap for a structured framework and optimized code for stable performance. Retina-ready support ensures images display clearly on high-resolution screens, and selective animations provide subtle interaction effects without cluttering the interface.This theme is suitable for educational websites seeking a structured, adaptable, and accessible design.

= Installation =
	
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in legacy-preschool-kindergarten in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' button to install and 'Activate' button to use.
